## 🚀 快速开始

### 第一步：安装依赖
```bash
pip install -r requirements.txt
```

### 第二步：配置（可选）
编辑 `config/config.json`，填入腾讯云API密钥（如需使用云端功能）

### 第三步：打包Android
```bash

```

### 第四步：安装到手机
```bash

```

## 🎓 摄影构图知识

### 三分法
将画面分为9个区域，主体放在交叉点上效果最佳。

### 黄金分割
按0.618比例分割画面，符合人眼审美习惯。

### 水平线位置
拍摄风景时，地平线放在上1/3或下1/3处，不要居中。

### 留白
给主体周围留出适当空间，增加画面呼吸感。
