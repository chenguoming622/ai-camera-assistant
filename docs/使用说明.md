# AI智能构图相机 - 使用说明

## 快速开始

### 1. 首次使用

1. 打开应用，授予相机权限
2. 相机预览自动启动
3. 屏幕上会显示：
   - 三分法网格线（白色）
   - 黄金分割线（白色虚线）
   - 水平参考线（黄色虚线）
   - 实时评分（左上角）
   - 构图建议（底部）

### 2. 拍摄流程

#### 步骤1：调整构图
- 移动手机，寻找最佳角度
- 观察屏幕上的实时建议
- 根据提示调整位置和角度
- 本地评分会实时更新

#### 步骤2：获取精准评分（可选）
- 点击左下角"AI精准评分"按钮
- 等待1-2秒
- 查看云端专业评分和详细建议
- 注意：每月有10000次免费额度

#### 步骤3：拍摄照片
- 当评分达到满意水平时
- 点击中间的📸按钮
- 照片自动保存到相册

### 辅助线说明

1. **三分法网格**（白色实线）
   - 将画面分为9个区域
   - 主体放在交叉点上效果最佳

2. **黄金分割线**（白色虚线）
   - 按0.618比例分割画面
   - 更专业的构图参考

3. **水平参考线**（黄色虚线）
   - 帮助保持画面水平
   - 避免拍摄倾斜

4. **主体标记框**（红色）
   - 自动识别画面主体
   - 显示主体位置和中心点

## 构图建议解读

### 位置建议
- "主体偏左，建议向右移" → 向右移动手机或调整拍摄角度
- "主体偏上，建议向下移" → 向下移动手机
- "主体位置符合黄金分割" → 位置很好，保持

### 水平建议
- "画面倾斜X度，请调整水平" → 旋转手机使其水平
- "水平良好" → 保持当前角度

### 光线建议
- "光线较暗，建议增加曝光" → 移到光线更好的位置
- "光线过亮，建议降低曝光" → 避免强光直射

### 评分建议
- "✓ 构图优秀，可以拍摄" → 8分以上，可以拍了
- "构图良好，可适当调整" → 6-8分，还能优化
- "建议调整构图" → 6分以下，需要改进

## 设置说明

点击右下角⚙按钮进入设置：

### 显示设置
- **显示网格线**：开关三分法网格
- **显示黄金分割**：开关黄金分割线
- **显示评分**：开关实时评分显示

### 云端API设置
- **启用腾讯云API**：开关云端精准评分功能
- **API Key**：输入您的腾讯云API密钥
- **API Secret**：输入您的腾讯云API密钥

## 常见问题

### Q: 为什么没有辅助线？
A: 检查设置中是否开启了"显示网格线"

### Q: AI精准评分按钮无法使用？
A: 
1. 检查是否配置了腾讯云API密钥
2. 检查网络连接
3. 确认API配额未用完

### Q: 评分一直很低怎么办？
A: 
1. 确保光线充足
2. 保持画面水平
3. 将主体放在网格交叉点附近
4. 避免画面过于杂乱

### Q: 照片保存在哪里？
A: 照片保存在手机相册的默认位置

## 摄影技巧

### 三分法构图
将主体放在画面的三分之一处，而不是正中央，会让照片更有张力。

### 黄金分割
更高级的构图方法，按0.618比例分割画面，符合人眼审美习惯。

### 留白
给主体周围留出适当空间，不要拍得太满。

### 水平线
拍摄风景时，将地平线放在上三分之一或下三分之一处，不要居中。

### 前景
加入前景元素可以增加画面层次感。
